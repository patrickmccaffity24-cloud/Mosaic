# Mosaic Policy Portal

A living public-facing policy/procedure knowledge base for Mosaic Youth Theatre of Detroit.

Admins upload PDF policy documents. Claude reads them and produces structured, plain-English versions with clause breakdowns. Approved policies are published to a public portal organized by department.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Supabase (Postgres, Auth, Storage)
- Anthropic Claude API (Sonnet 4)
- Vercel (deployment)
